# 콘텐츠 작성 가이드

## 폴더 구조
```
content/
  drafts/          ← 여기에 마크다운으로 원고 작성
    _template.md   ← 복사해서 사용하는 템플릿
    my-article.md  ← 작성 중인 원고
```

## 카테고리 코드
| 코드 | 라벨 | 설명 |
|------|------|------|
| `strategy` | 입시전략 | 입시 전략, 대입 분석 |
| `column` | 교육칼럼 | 교육 트렌드, 학습법 |
| `success` | 합격스토리 | 합격 사례, 후기 |
| `news` | 공지사항 | 학원 소식, 일정 |

## 작성 완료 후
Claude Code에게 "이 글 완성이야, UI로 보내줘"라고 말하면
자동으로 `.queue/`에 작업 요청이 생성됩니다.
